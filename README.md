## Important distinction (baked into the bonus)
The **=COPILOT() worksheet function is vanilla**: on recalc it sees only your prompt text plus the cell ranges you pass it. Personalization, Workbook Rules, Skills, and reference files shape the **Copilot chat/agent** experience, not the function. So the bonus demos use the Copilot **chat** on the separate **Sales-by-Region** dataset, and the Part 3 function demo inlines its rules directly in the prompt.
