# AI in Excel for Finance & Accounting Teams — IOFM Webinar (Aug 10, 2026)

Materials for the IOFM webinar. Five core parts (~45 min) plus a flexible bonus section and Q&A.

## Instructor/  (what George uses to run the session)
- **AI-in-Excel-for-Finance-IOFM.pptx** — 21-slide deck with read-aloud speaker notes; the optional bonus section includes two before/after demos
- **Instructor-Run-of-Show.docx** — minute-by-minute plan with pacing guidance, a file map, setup checklist, prompts, short/long demo versions, and demo fallbacks
- **data-solutions.xlsx** — answer key (cleaned register, extracted memos with policy-grounded flags, COPILOT() formula reference)

## Extras/  (attendee take-homes / lagniappes)
- **Follow-Along-Guide.docx** — idiotproof, student-facing step-by-step for the live demos (which file, which tab, which prompt), including the bonus before/after
- **Prompt-Cheat-Sheet.docx** — copy-paste Copilot and =COPILOT() prompts
- **Which-AI-Tool-When.docx** — tool comparison + deterministic vs probabilistic guide
- **Make-Copilot-Work-Like-Your-Team.docx** — custom instructions vs Skills vs reference files, with before/after exercises
- **Guardrails-and-Audit-Trail-Checklist.docx** — reviewer checklist + audit-trail log
- **First-3-Workflows-Quickstart.docx** — the three workflows to try first
- **data-start.xlsx** — hands-on sample workbook (InvoiceRegister, ExpenseMemos, APMonthly)

### Extras/Reference-Files/  (demo inputs for the bonus section)
- **Expense-and-Purchasing-Policy.docx** — sample policy with hard thresholds; attach as a Copilot reference file to ground the Policy Flag output
- **Cost-Center-Codebook.xlsx** — maps cost-center codes to names/approvers, plus the approved expense categories

## Bonus: the before/after
- **Custom instructions (slide 17)** — set one standing instruction on APMonthly and re-run the summary; the *delivery* changes (bullets, no cents, Favorable/Unfavorable).
- **Reference file (slide 18)** — attach the expense policy on ExpenseMemos and re-run the Policy Flag prompt; the *result* changes (flags cite the real $75 meal and $100 gift limits).

## Timing
Core content runs 2:00–2:48 (5 + 12 + 12 + 10 + 6 min). The bonus section (slides 15–19) is optional: expand it if the demos run fast, skip it if you're behind. See the run-of-show for short/long versions of each demo.
